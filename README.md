# MemSpy

Console utility to find footprints of sensitive data in process memory.

Usage:
    MemSpy.exe [/s=...] [/f=...] [/l [/i=...] [/t=...]]
        /s - searching secret
        /f - process file name
        /l - search in loop
        /i - number of loop iterations
        /t - sleep timeout in milliseconds after each loop
